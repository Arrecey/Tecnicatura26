## Comandos vistos
git init
Inicializa un repositorio Git en la carpeta actual.

git status
Muestra el estado de los archivos del proyecto.

git add .
Agrega todos los archivos al área de preparación

git commit -m "mensaje"
Guarda los cambios realizados.

touch finales.py
Creamos el archivo finales

python -m venv venv
venv/scripts/activate
Creamos el entorno virtual y lo activamos

git branch -M main
Cambiamos el nombre de master a main

git remote add origin URL
Vincula el repositorio local con un repositorio remoto.

git push origin main
Envía los cambios al repositorio remoto.
